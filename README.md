# AI + 半导体产业链投研知识库

本仓库沉淀一套面向投资顾问使用的 AI + 半导体产业链研究框架。内容从行业总图、AI 相关芯片、半导体制造、设备、材料、商业模式、投资分析、全球标的池和术语体系展开。

## 研究目标

把半导体产业链从“概念罗列”整理成一套可用于客户讲解、行业研究和标的跟踪的投研手册。核心不是记住名词，而是建立判断链：

需求在哪里 → 瓶颈在哪里 → 利润在哪里 → 扩产在哪里 → 风险在哪里。
