# 7.2 瓶颈判断

半导体投资中，最赚钱的环节往往不是需求最大的环节，而是供给最紧、认证最难、替代最慢的环节。

## 1. 什么是瓶颈

瓶颈是限制产业扩张的关键环节。它可能是 GPU、HBM、CoWoS、先进制程、光刻机、封装基板、测试设备、电力、液冷、光模块或高端 PCB。

## 2. AI 链条瓶颈

AI 当前典型瓶颈包括 GPU/ASIC、HBM、先进封装、ABF/FC-BGA、CoWoS、光模块、高速交换、数据中心电力和液冷。

## 3. 设备材料瓶颈

在半导体制造中，光刻、刻蚀、沉积、量测检测、硅片、电子特气、光刻胶、CMP 材料和封装基板都可能成为阶段性瓶颈。

## 4. 判断方法

看交期是否拉长、价格是否上涨、客户是否抢产能、扩产是否困难、认证是否漫长、替代是否稀缺。如果只是需求增长但供给容易扩张，就不一定有超额利润。

一句话总结：

**瓶颈环节拥有更强定价权，投资半导体要先找到产业链中真正卡脖子的地方。**
