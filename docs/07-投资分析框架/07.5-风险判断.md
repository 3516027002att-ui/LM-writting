# 7.5 风险判断

半导体行业高成长、高波动、高估值，风险判断必须放在投资框架里。

## 1. 需求错配

市场预期 AI 需求持续高增长，但实际订单可能低于预期。云厂商 CapEx、模型商业化、推理成本和应用 ROI 都可能影响需求。

## 2. 供给错配

短期供不应求会刺激扩产，长期可能变成供过于求。硅片、存储、面板、部分材料都有明显周期属性。

## 3. 技术路线错配

技术路线可能变化，例如 GPU 与 ASIC、可插拔光模块与 CPO、ABF 与玻璃基板、风冷与液冷、不同封装路线之间都可能出现切换。

## 4. 估值错配

好公司如果估值过高，未来业绩只要低于预期也会下跌。半导体投资要区分产业好、公司好和股价好。

## 5. 地缘与供应链风险

半导体受出口管制、地缘政治、供应链安全、客户迁移和国产替代政策影响明显。投资时要同时看机会和限制。

一句话总结：

**半导体投资最大的风险，不是看错行业，而是看对行业却买在过高预期、错误环节或错误周期。**
