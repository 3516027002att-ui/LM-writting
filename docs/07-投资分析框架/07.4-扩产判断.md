# 7.4 扩产判断

扩产是半导体投资的重要领先指标。扩产代表景气，也可能埋下未来供给过剩的种子。

## 1. 看谁扩产

云厂商扩数据中心，拉动服务器、GPU、网络、电源、液冷。晶圆厂扩产，拉动设备、材料、洁净室、厂务系统。封装厂扩先进封装，拉动封装基板、测试、键合、临时键合、RDL。

## 2. 看扩什么

扩成熟制程和扩先进制程意义不同。扩传统封装和扩 CoWoS 不同。扩普通 PCB 和扩 AI 高多层 PCB 不同。

## 3. 看什么时候释放

扩产从公告到设备进场、产线调试、良率爬坡、客户认证和收入确认，需要时间。投资中不能把公告立即等同于利润。

## 4. 看是否形成过剩

当所有公司都扩产，未来可能从短缺变成过剩。周期品和材料品尤其要警惕价格下行。

一句话总结：

**扩产短期是订单，长期是供给；优秀投研要同时看到扩产带来的机会和未来过剩风险。**
