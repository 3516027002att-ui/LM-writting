# 7.1 需求判断

需求判断是半导体投资的第一步。没有需求，任何扩产、国产替代、技术升级都很难转化为持续业绩。

## 1. 需求来源

半导体需求主要来自 AI、消费电子、汽车电子、工业、通信、数据中心、存储周期和库存周期。不同需求对应不同产业链环节。

## 2. AI 需求

AI 训练需求看大模型规模、云厂商 CapEx、GPU/ASIC 订单、HBM 和先进封装。AI 推理需求看应用调用量、单位 token 成本、推理服务器和 ASIC 部署。

## 3. 周期需求

消费电子和存储常有强周期属性。库存去化、价格反弹、产能利用率提升、订单恢复都可能带来周期修复。

## 4. 结构性需求

AI、汽车电子、功率半导体、国产替代、先进封装、高速互连属于更长期的结构性需求，但也会有阶段性高估和回调。

一句话总结：

**需求判断要先分清是周期修复还是结构增长，再沿产业链判断哪些环节真正受益。**
