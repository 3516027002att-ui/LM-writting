# 7.3 利润判断

收入增长不等于利润增长。投资半导体必须判断利润最终留在哪个环节。

## 1. 定价权

有定价权的公司更容易保住利润。定价权来自稀缺性、客户粘性、技术壁垒、生态锁定和认证难度。

## 2. 成本结构

设计公司研发费用高但毛利率通常高。制造公司资本开支和折旧高。设备材料公司看研发和认证。封测公司传统业务利润率较低，先进封装利润弹性更好。

## 3. 规模效应

设备和材料公司进入客户后，规模放量能摊薄固定成本。晶圆厂稼动率上升会显著改善利润。封测厂先进封装产能利用率是关键。

## 4. 利润陷阱

服务器整机收入可能暴涨但利润率不高。材料公司产品很多但高端占比低。设备公司订单多但验收慢。设计公司销售强但库存和价格压力可能拖累利润。

一句话总结：

**利润判断的核心是看谁有定价权、谁能留住毛利、谁的增长不会被成本和竞争吃掉。**
