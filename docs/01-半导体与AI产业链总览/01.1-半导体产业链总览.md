# 1.1 半导体产业链总览

半导体产业链可以理解为一条从需求到交付的完整链条：终端需求提出需要什么芯片，芯片设计公司定义芯片怎么工作，EDA/IP 帮助完成设计，晶圆厂把设计变成电路，设备材料提供制造能力，封测厂把晶圆变成可使用芯片，最终进入终端应用。

一句话理解：

**半导体产业链 = 设计芯片 → 制造晶圆 → 封装测试 → 进入终端。**

## 1. 六个核心环节

1. 芯片设计：决定芯片要干什么。典型公司包括 GPU、CPU、ASIC、存储控制器、网络芯片、模拟芯片设计公司。
2. EDA/IP：决定芯片能不能被设计出来。EDA 是设计工具，IP 是可复用模块。
3. 晶圆制造：决定芯片能不能被造出来。核心看制程、良率、产能、客户信任和资本开支。
4. 设备材料：决定制造能力上限。没有设备材料，先进制程和先进封装都无法落地。
5. 封装测试：把裸片变成可交付产品。AI 时代先进封装从后道加工升级为系统性能环节。
6. 终端应用：决定需求来源。AI、消费电子、汽车、工业、数据中心共同决定景气周期。

## 2. 投资视角

芯片设计公司的壁垒在产品定义、架构、软件生态、客户绑定和迭代速度。晶圆制造公司的壁垒在资本开支、制程平台、良率和客户信任。设备材料公司的壁垒在认证周期、工艺绑定和国产替代难度。封测公司的壁垒在先进封装能力、产能、良率和客户订单。

## 3. 用一颗 AI GPU 串起产业链

一颗 AI GPU 先由设计公司完成架构和电路设计，再通过 EDA 工具完成验证和流片，晶圆厂用先进制程制造大芯片，设备公司提供光刻、刻蚀、沉积、CMP、检测等设备，材料公司提供硅片、光刻胶、电子特气、湿电子化学品、靶材等材料，封装厂用 CoWoS、2.5D、HBM 等先进封装把 GPU 和 HBM 集成，最终进入 AI 服务器和数据中心。

## 4. 投资顾问要具备的能力

看到一家公司，要判断它在产业链中的位置。看到一个技术词，要知道影响哪个环节。看到一条新闻，要判断传导方向。看到业绩变化，要区分是周期修复还是结构性增长。

一句话总结：

**半导体不是一家公司或一种芯片，而是一整套从需求、设计、制造、设备材料、封装测试到终端应用的工业体系。**
