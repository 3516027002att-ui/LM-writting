# 6.3 设备材料公司的商业模式

设备材料公司是半导体制造能力的上限。晶圆厂扩产、先进制程推进和先进封装升级，都要先经过设备材料公司的认证和供货。

## 1. 设备公司的商业模式

设备公司收入来自新设备销售、备件、耗材、服务和升级。先进设备客户认证周期长，一旦进入客户产线，后续服务和复购粘性较强。

## 2. 材料公司的商业模式

材料公司收入来自持续消耗。硅片、电子特气、湿电子化学品、光刻胶、靶材、CMP 材料、封装材料都是复购型业务。材料的关键是稳定性、纯度、批次一致性和客户认证。

## 3. 壁垒

设备壁垒在工程能力、工艺适配、客户验证、控制系统、零部件供应和服务能力。材料壁垒在配方、纯度、缺陷控制、客户工艺绑定和供应稳定性。

## 4. 国产替代逻辑

国产替代不只是“国产公司做出来”，而是要经过客户验证、稳定供货、批量导入和工艺持续优化。真正进入产线后，客户切换成本高，粘性也高。

## 5. 投资重点

设备看订单、交付、验收、毛利率、研发费用和客户结构。材料看产品品类、客户认证、产能利用率、价格周期、毛利率和高端产品占比。

一句话总结：

**设备材料公司赚的是制造能力的钱，最强的不是概念最多的公司，而是能稳定进入客户产线并持续复购的公司。**
