# 6.1 芯片设计公司

芯片设计公司负责定义芯片要解决什么问题、用什么架构、服务什么客户和生态。它们通常采用 Fabless 模式，不自建晶圆厂，而是把制造交给台积电、三星、联电、中芯国际等晶圆代工厂。

## 1. 商业模式

芯片设计公司的核心是研发和产品定义。收入来自芯片销售、平台方案、软件生态、IP 授权和系统方案。毛利率通常高于制造和封测，但研发投入也很高。

## 2. 壁垒

核心壁垒包括架构能力、软件生态、客户绑定、产品迭代、先进制程协同、供应链资源和系统方案能力。AI 芯片尤其看软件栈、编译器、库、开发者生态和集群调优。

## 3. AI 时代变化

GPU 公司从卖芯片升级为卖算力平台，ASIC 公司从硬件公司升级为云厂商基础设施的一部分。推理芯片公司要面对成本、功耗、生态和部署便利性竞争。

## 4. 投资重点

看市场空间、客户结构、产品代际、生态壁垒、毛利率、研发费用率、库存和代工产能。AI 芯片还要看 HBM 供应、先进封装产能和整机生态。

一句话总结：

**芯片设计公司赚的是架构、生态和产品定义的钱，AI 时代最强设计公司往往不只卖芯片，而是卖完整算力平台。**
