# 8.3 韩股标的池

## 1. 存储与 AI 链条

Samsung Electronics：存储、晶圆制造、封装、HBM 和 AI 产业链核心。  
SK Hynix：HBM、DRAM、NAND，AI 存储主线最强之一。

## 2. 设备

Wonik IPS、Jusung Engineering、Eugene Technology、TES：沉积设备和存储设备链。  
KCTech：CMP 设备、湿法清洗、CMP slurry。  
PSK：dry strip、dry cleaning。  
Hanmi Semiconductor：HBM / TC Bonder 核心。  
Techwing、Exicon、UniTest：存储测试和分选。

## 3. 材料

Wonik Materials：电子特气核心。  
Foosung：含氟特气。  
TEMC：电子特气成长型。  
Soulbrain：湿化学品、CMP slurry、清洗刻蚀材料。  
ENF Technology：湿法工艺化学品。  
Dongjin Semichem：光刻胶、CMP slurry、光刻配套材料。  
Hansol Chemical：高纯双氧水和电子化学品。  
Samyang NC Chem：光刻胶核心原料。

## 4. 封装基板与 PCB

Samsung Electro-Mechanics：FC-BGA 和 AI 服务器封装基板核心。  
LG Innotek：FC-BGA 成长型标的。  
Simmtech：存储封装基板和 memory module PCB。  
ISU Petasys：AI 服务器和网络设备高多层 PCB 第一核心。  
Daeduck Electronics、Korea Circuit：封装基板和 PCB 补充。  
SKC：玻璃基板方向。

一句话总结：

**韩股核心围绕存储、HBM、三星/SK 海力士供应链、封装基板、测试和电子材料展开。**
