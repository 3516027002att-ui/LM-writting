# 8.5 范围外但必须知道

本知识库默认覆盖 A 股、美股、韩股、日股，但全球半导体产业链中有一些公司不在这些市场，却必须作为产业参照。

## 1. 台股

TSMC：全球先进晶圆代工和先进封装核心。  
ASE / 日月光：全球封测核心。  
Unimicron、Kinsus、Nan Ya PCB：ABF / IC 载板和 PCB 重要公司。  
GlobalWafers：硅片核心。  
Elite Material、ITEQ：高速 CCL 重要标的。  
欣兴、华通、健鼎、臻鼎、南电：PCB、IC 载板、HDI 和高端板核心公司。

## 2. 欧洲

ASM International：ALD 核心设备。  
Besi：Die Attach、Hybrid Bonding 核心设备。  
ASML：光刻机绝对核心，虽有美股 ADR，但主上市在欧洲。  
Merck KGaA：电子材料、特气、光刻材料。  
BASF：湿电子化学品和 process chemicals。  
Siltronic：硅片。  
AT&S：先进 IC substrate。  
Umicore：贵金属和薄膜材料参考。

## 3. 未上市或已退市

JSR：全球光刻胶核心公司，已私有化退市。  
Shinko Electric：日本封装基板和封装产业重要公司，已退市。  
Plansee：难熔金属靶材重要公司。  
TANAKA：贵金属材料和靶材重要公司。  
SEMES：韩国半导体设备重要公司，未独立上市。  
SK Specialty、SK Siltron：韩国特气和硅片核心资产，但交易映射不纯。  
珠海越亚：国内 IC 封装载板重要公司，IPO 观察。

一句话总结：

**范围外公司不是可直接交易核心池，但它们决定产业真实格局，做全球半导体研究必须知道。**
