# 8.4 日股标的池

## 1. 设备

Tokyo Electron：涂胶显影、刻蚀、沉积、清洗等综合设备核心。  
SCREEN：晶圆清洗设备核心。  
Kokusai Electric：沉积设备，存储链条强。  
Ebara：CMP 设备核心。  
Advantest：测试设备龙头。  
Tokyo Seimitsu：探针台和划片。  
DISCO：划片、减薄、研磨。  
TOWA：塑封设备。  
Lasertec：EUV 掩膜检测核心。  
Hitachi High-Tech：CD-SEM 和量测。

## 2. 材料

Shin-Etsu、SUMCO：硅片。  
Nippon Sanso、Kanto Denka、Resonac、Air Water：电子气体和特气。  
Fujifilm、Stella Chemifa、Mitsubishi Gas Chemical、Tokuyama：湿电子化学品。  
JX Advanced Metals、Tosoh、ULVAC、Sumitomo Metal Mining：靶材。  
Tokyo Ohka Kogyo、Shin-Etsu、Fujifilm、Sumitomo Chemical：光刻胶。  
Fujimi、Resonac、Fujifilm、AGC、Kuraray：CMP 材料。

## 3. 封装基板与 PCB

Ibiden：高端 ABF / FC-BGA 封装基板核心。  
Ajinomoto：ABF 膜核心材料。  
TOPPAN：FC-BGA 封装基板。  
Kyocera：有机封装基板和陶瓷封装。  
Sumitomo Bakelite：封装基板材料。  
Meiko Electronics：PCB 制造核心。  
CMK：汽车和高可靠 PCB。  
Panasonic：高速低损耗 PCB 材料。

一句话总结：

**日股半导体强在设备、材料和封装基板，尤其是 TEL、SCREEN、Advantest、Shin-Etsu、TOK、Ibiden、Ajinomoto 等核心公司。**
