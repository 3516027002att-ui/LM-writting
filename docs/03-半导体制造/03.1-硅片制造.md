# 3.1 硅片制造

硅片是芯片制造的起点。后续光刻、刻蚀、沉积、离子注入、CMP 都建立在硅片上。硅片质量越高，后续工艺的良率上限越高。

## 3.1.1 从沙子到电子级硅

硅来自石英砂，但不能直接用于芯片制造。石英砂先变成工业硅，再经过化学提纯成为高纯多晶硅。半导体需要极高纯度，杂质会带来漏电、击穿、性能不稳定和良率下降。

投资角度看，电子级硅材料、电子级硅片、高纯多晶硅和晶圆厂开工率相关。它们具有周期属性，也受长期订单和客户认证影响。

## 3.1.2 拉制单晶硅

多晶硅需要拉制成单晶硅。单晶硅的晶体排列统一，适合制造稳定的晶体管结构。拉晶过程要控制温度、拉速、旋转、氧含量、杂质、晶体直径和缺陷。

300mm 晶圆是先进逻辑和存储主流规格，200mm 多用于成熟制程、功率、模拟、MEMS 等方向。

## 3.1.3 切片与抛光

晶棒要经过切片、倒角、磨削、抛光、清洗和检测，变成足够薄、足够平、足够洁净、缺陷足够少的晶圆。表面缺陷会影响光刻、沉积、刻蚀和后续良率。

一句话总结：

**电子级硅解决纯度，单晶硅解决晶体结构，切片抛光解决形状、平整度和表面质量。**
