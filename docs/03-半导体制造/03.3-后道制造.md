# 3.3 后道制造

前道完成后，晶圆上的裸片还不能直接使用。后道制造负责筛选、减薄、切割、封装和测试，把裸片变成可以交付给客户的芯片。

## 3.3.1 晶圆测试 CP

晶圆测试在切割前进行，用探针台和测试机接触裸片焊盘，判断每颗裸片是否合格，并形成晶圆图。AI 芯片价值高，越早筛出坏 die，越能减少后续封装浪费。

## 3.3.2 晶圆减薄与切割

晶圆减薄有利于薄封装、散热和高密度集成。HBM、先进封装、3D 堆叠尤其依赖薄晶圆处理。切割要避免崩边、颗粒、应力和损伤。

## 3.3.3 传统封装方式

传统封装负责保护芯片、连接外部电路、提供散热和可靠性。常见形式包括引线框架、QFN、BGA、Flip-Chip 等。传统封装不等于低端，不同芯片需要不同封装。

## 3.3.4 成品测试 FT

封装后还要进行最终测试，包括电性、功能、性能、功耗、温度和可靠性。AI GPU、HBM、交换芯片测试复杂度高，测试时间和成本都较高。

一句话总结：

**后道制造负责把晶圆变成可交付芯片，AI 时代后道从加工环节升级为系统集成和交付能力。**
