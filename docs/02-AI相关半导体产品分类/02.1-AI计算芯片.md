# 2.1 AI 计算芯片

AI 计算芯片是大模型训练和推理的核心硬件。它们负责执行矩阵乘法、张量计算、向量运算、数据调度和模型推理。

## 1. CPU

CPU 是通用处理器，擅长复杂逻辑、系统调度和控制。它灵活、生态成熟，但并行计算能力不如 GPU。AI 系统中，CPU 负责调度、数据预处理、系统管理和与其他芯片协同。

## 2. GPU

GPU 是当前 AI 训练和高性能推理的主力。它擅长大规模并行矩阵计算，适合训练大模型和运行大规模推理。GPU 的壁垒不仅在硬件，还在软件生态、CUDA、库、集群调度和客户习惯。

## 3. AI ASIC

AI ASIC 是专用 AI 芯片，针对训练或推理任务优化。它的优势是能效高、成本可控、云厂商可自定义；缺点是通用性较差，研发周期长，生态建设难。典型路线包括 TPU、Trainium、Inferentia 和各类自研推理芯片。

## 4. FPGA

FPGA 是可编程硬件芯片，适合低延迟、可重配置和特定场景加速。它灵活但开发难度高，性能和能效一般不如 ASIC。AI 中多用于特定推理、边缘计算、金融交易、网络加速等场景。

## 5. NPU

NPU 是神经网络处理器，常见于端侧、边缘和部分数据中心推理场景。它低功耗、适合特定神经网络推理，不适合大规模训练。

## 6. DPU / SmartNIC

DPU 和智能网卡负责网络、存储、安全、虚拟化和数据搬运。AI 集群中，它们能减轻 CPU 负担，提高网络效率和系统利用率。

一句话总结：

**CPU 管调度，GPU 管通用并行算力，ASIC 管专用高能效，FPGA 管灵活可编程，NPU 管神经网络推理，DPU 管数据搬运和网络卸载。**
