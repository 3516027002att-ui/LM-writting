# 2.3 AI 互连芯片

AI 集群不是一台服务器，而是成千上万张加速卡协同工作。互连芯片决定这些 GPU/ASIC 能不能像一个整体一样运行。

## 1. 高速网卡芯片

高速网卡负责服务器与网络之间的数据传输。AI 集群需要高带宽、低延迟、低丢包和拥塞控制，网卡和 SmartNIC/DPU 价值提升。

## 2. 交换芯片

交换芯片是数据中心网络的核心。GPU 集群规模越大，对交换芯片带宽、端口数量、SerDes 速率和功耗要求越高。800G、1.6T 交换机会推动高速 PCB、光模块和 CPO 发展。

## 3. SerDes

SerDes 是高速串行收发器，是芯片之间、板级之间、模块之间高速通信的基础。AI 时代 SerDes 速率不断提升，对信号完整性、功耗和封装提出更高要求。

## 4. 光通信芯片

包括 DSP、激光器驱动、TIA、硅光芯片、调制器、光探测器等。GPU 集群越大，光互连越重要。

## 5. GPU 高速互连相关芯片

包括 NVLink、PCIe、CXL、交换芯片和桥接芯片等。它们决定服务器内部 GPU 之间、CPU 与 GPU 之间、内存与加速器之间的通信效率。

一句话总结：

**AI 互连的本质是把很多昂贵芯片连接成一个高效集群，网络慢就等于算力浪费。**
