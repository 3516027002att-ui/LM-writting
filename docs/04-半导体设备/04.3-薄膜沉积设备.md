# 4.3 薄膜沉积设备

沉积负责把材料铺到晶圆上。光刻定义图案，刻蚀去掉材料，沉积则负责形成绝缘层、导电层、阻挡层、介质层、金属层和外延层。

主要路线包括 CVD、PVD、ALD、外延设备。

## 1. A 股

北方华创：平台型设备龙头，PVD、CVD、ALD 等产品线重要。

拓荆科技：A 股薄膜沉积设备最纯核心之一，覆盖 PECVD、ALD、SACVD、HDPCVD、Flowable CVD 等。

微导纳米：ALD 方向重要标的，但半导体业务弹性要结合光伏收入结构观察。

中微公司：主标签是刻蚀，但 MOCVD 设备布局用于 LED、化合物半导体、功率电子等。

## 2. 美股

Applied Materials：全球薄膜沉积设备第一梯队，CVD、PVD、ALD、外延均强。

Lam Research：刻蚀龙头，同时有 PECVD、HDP-CVD、ALD、钨金属化等沉积产品。

Veeco：MOCVD、PVD、ALD、化合物半导体、光电子设备相关。

ASM International：全球 ALD 核心龙头，主上市在欧洲，美股 OTC 可观察。

## 3. 韩股

Wonik IPS、Jusung Engineering、Eugene Technology、TES 都是韩国薄膜沉积设备观察标的，主要看 DRAM、NAND、HBM 和韩国晶圆厂投资。

## 4. 日股

Tokyo Electron：沉积设备核心龙头之一。

Kokusai Electric：CVD、ALD 批量沉积设备，存储链条强。

ULVAC：真空和 PVD 相关设备补充。

一句话总结：

**薄膜沉积设备全球核心看 AMAT、TEL、Kokusai、Lam；A 股看拓荆科技、北方华创、微导纳米。**
