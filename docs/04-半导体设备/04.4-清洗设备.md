# 4.4 清洗设备

清洗不是简单洗干净，而是在关键工艺之间降低缺陷、保护良率。先进制程越复杂，晶圆越不能容忍颗粒、金属、有机残留和腐蚀副产物。

## 1. A 股

盛美上海：A 股清洗设备第一核心，覆盖单片、槽式清洗、电镀、先进封装湿法等。

北方华创：平台型设备龙头，清洗为重要产品线之一。

至纯科技：湿法工艺设备和高纯工艺系统相关。

江化微、上海新阳、晶瑞电材：更多偏湿电子化学品和清洗液材料。

## 2. 美股

ACM Research：美股清洗设备核心标的，与盛美上海有产业映射关系。

Lam Research：strip/clean 相邻工艺重要。

Entegris：高纯材料、过滤、污染控制和 post-CMP 材料。

## 3. 韩股

KCTech：CMP/Wet Cleaning System 和 CMP slurry。

PSK：dry strip / dry cleaning。

Soulbrain：Etch & Cleaning、HF/BOE、CMP slurry 等材料。

SEMES：韩国清洗设备重要公司，但未独立上市。

## 4. 日股

SCREEN：全球晶圆清洗设备核心龙头。

Tokyo Electron：综合设备龙头，单片清洗布局。

J.E.T.：日股清洗设备专业标的。

Fujifilm、MGC、Tokuyama：清洗相关高纯化学品和湿电子化学品。

一句话总结：

**清洗设备全球核心看 SCREEN、TEL、盛美/ACM；清洗材料看江化微、Fujifilm、MGC、Tokuyama、Entegris、Soulbrain。**
