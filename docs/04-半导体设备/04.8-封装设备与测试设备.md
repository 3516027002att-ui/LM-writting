# 4.8 封装设备与测试设备

封装设备负责把裸片变成可用芯片，测试设备负责判断芯片能不能稳定工作。AI 时代，HBM、Chiplet、CoWoS、先进封装推动后道设备价值量上升。

## 1. 设备类型

封装设备包括贴片、固晶、Die Bonder、Flip-Chip Bonder、TC Bonder、Hybrid Bonding、Wire Bonder、塑封、划片、减薄等。

测试设备包括测试机 ATE、分选机 Handler、探针台 Prober、探针卡 Probe Card、老化测试 Burn-in、测试插座等。

## 2. A 股

长川科技：测试设备平台核心，测试机、分选机、探针台、AOI。

华峰测控：ATE 测试机核心，模拟、数模混合、功率器件测试。

金海通：测试分选机纯标的。

精智达：存储测试、老化修复、算力芯片测试观察。

联动科技：功率半导体测试设备。

新益昌、耐科装备、文一科技：固晶、塑封、封装模具等补充。

## 3. 美股

Teradyne：ATE 测试机龙头之一。

Cohu：测试自动化、分选机、测试接口。

FormFactor：探针卡和晶圆级测试，HBM/先进封装相关。

Kulicke & Soffa：Wire Bonder、Die Attach、Flip Chip、TCB。

Besi：Die Attach、Hybrid Bonding 全球核心，主上市欧洲，美股 OTC。

Aehr Test Systems：Burn-in、晶圆级老化测试。

## 4. 韩股

Hanmi Semiconductor：HBM / TC Bonder 核心标的。

Techwing：Memory Test Handler、HBM Cube Prober。

Exicon、UniTest：存储测试、Burn-in。

Protec、Mirae：Die Bonder、Handler、后道设备补充。

## 5. 日股

Advantest：全球 ATE 测试设备龙头之一。

Tokyo Seimitsu / ACCRETECH：探针台和划片。

TOWA：塑封设备核心。

Shibaura Mechatronics：Flip Chip Bonder、先进封装键合设备。

DISCO：划片、减薄、研磨、package singulation。

Tazmo：临时键合/解键合，先进封装薄晶圆处理。

一句话总结：

**测试机看 Advantest、Teradyne、长川、华峰；键合设备看 Hanmi、Besi、KLIC、Shibaura；塑封看 TOWA；切割减薄看 DISCO。**
