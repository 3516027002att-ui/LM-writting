# 4.6 CMP 设备

CMP 是化学机械抛光，用化学反应和机械摩擦把晶圆表面重新磨平。多层互连、3D NAND、DRAM、HBM 和先进封装都离不开平坦化。

CMP 设备负责执行抛光动作，CMP 材料包括抛光液、抛光垫和清洗液。

## 1. A 股

华海清科：A 股 CMP 设备第一核心标的，产品覆盖 CMP 设备、减薄、划片、湿法、膜厚量测、晶圆再生等。

安集科技：CMP 抛光液核心标的。

鼎龙股份：CMP 抛光垫核心标的。

## 2. 美股

Applied Materials：美股 CMP 设备核心，综合设备龙头。

Entegris：通过 CMC Materials 体系覆盖 CMP slurry / pad、过滤、污染控制、post-CMP 材料。

Qnity：CMP pads 和 CMP slurries 核心材料标的。

## 3. 韩股

KCTech：CMP 设备 + CMP slurry，韩国存储链条相关。

Soulbrain、Dongjin Semichem：CMP slurry 和电子材料。

## 4. 日股

Ebara：日股 CMP 设备第一核心，全球 CMP 设备龙头之一。

Fujimi：CMP slurry 和精密研磨材料核心。

Resonac、Fujifilm、AGC、Kuraray：CMP slurry、post-CMP cleaner、CMP pad 相关。

一句话总结：

**CMP 设备看 AMAT、Ebara、华海清科；CMP 抛光液看 Fujimi、安集、Entegris、Soulbrain；CMP 抛光垫看 Qnity、鼎龙。**
