# 5.8 PCB

PCB 是 Printed Circuit Board，中文叫印制电路板。封装基板连接芯片和封装外部，PCB 则连接服务器、交换机、加速卡、主板、内存、电源、网卡和光模块。

一句话理解：

**封装基板是芯片封装里的连接层，PCB 是服务器、交换机和加速卡里的系统级连接骨架。**

## 1. 产品分类

高多层板 / HLC：AI 服务器、交换机、路由器、GPU 加速卡、UBB、CPU 主板核心。  
HDI 板：通过盲孔、埋孔、微孔和细线路实现高密度互连。  
高速板 / 高频板：用于 800G/1.6T 交换机、AI 服务器、光模块、路由器。  
背板 / Midplane / UBB：AI 服务器和高速交换设备核心互连板。  
FPC / 刚挠结合板：消费电子、汽车、传感器、部分服务器和光模块内部连接。  
CCL / 覆铜板：PCB 上游核心材料，高速低损耗材料在 AI 服务器中价值提升。

AI 传导：AI GPU/ASIC、HBM、CPO、800G/1.6T 交换机、AI 服务器扩张 → OAM、UBB、CPU 主板、交换板、存储板、网卡板、光模块板需求提升 → 高多层、HDI、高速低损耗 PCB 价值量提升。

## 2. A 股

沪电股份：A 股 AI 服务器和高速网络 PCB 核心标的。

胜宏科技：AI 高阶 HDI / 高多层 PCB 核心标的，AI 服务器弹性强。

深南电路：PCB + 封装基板 + 电子装联平台龙头。

生益电子：AI 服务器 PCB、OAM、UBB 纯度很高。

鹏鼎控股：消费电子 PCB 巨头，AI 服务器 HDI / HLC 加速切入。

景旺电子：汽车 PCB 强，AI 高阶 HDI / HLC 是第二增长曲线。

奥士康：AI 服务器 PCB 补充标的。

生益科技：PCB 上游 CCL 核心标的，高速低损耗材料主线。

## 3. 美股

TTM Technologies：美股先进 PCB 核心标的，AI、云计算、数据中心和高端计算相关。

Jabil、Sanmina：EMS / PCBA / 系统制造观察，不是纯 PCB 制造商。

## 4. 韩股

ISU Petasys：韩股 AI 服务器和网络设备高多层 PCB 第一核心标的。

Simmtech：存储 PCB、memory module PCB、AI 内存链条。

Daeduck Electronics、Korea Circuit：更偏封装基板和 PCB 补充观察。

## 5. 日股

Meiko Electronics：日股 PCB 制造核心标的，覆盖 HDI、高频、高散热、多层板。

CMK：汽车和高可靠 PCB 标的，AI 服务器纯度不高。

NOK / MEKTEC：FPC / 柔性线路板间接观察。

Ibiden：更偏高端封装基板，属于广义 PCB 高端延伸。

Panasonic：高速低损耗 PCB 材料，MEGTRON 系列重要。

## 6. 范围外但必须知道

台股 PCB/CCL 非常重要。欣兴、华通、健鼎、臻鼎、南电是全球 PCB 和 IC 载板重要公司。台光电、联茂是 AI 服务器高速 CCL 重要标的。

一句话总结：

**AI PCB 主线看高多层、HDI、高速低损耗材料；A 股看沪电、胜宏、深南、生益电子、生益科技；美股看 TTM；韩股看 ISU Petasys；日股看 Meiko 和 Panasonic；台股必须重点知道欣兴、台光电、联茂等。**
