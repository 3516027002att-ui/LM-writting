# 5.5 光刻胶

光刻胶是光刻工艺里的核心耗材。光刻机负责用光照出图案，光刻胶负责接住这个图案。

## 1. 产品分类

g/i 线光刻胶：成熟制程、功率器件、模拟芯片、分立器件、传感器、封装。  
KrF 光刻胶：248nm DUV，用于成熟到中高端节点、存储、特色工艺。  
ArF 光刻胶：193nm DUV，分干法和浸没式，是先进制程和高端存储重要材料。  
EUV 光刻胶：13.5nm，服务最先进逻辑和先进存储。  
先进封装光刻胶 / PSPI：用于 RDL、凸点、电镀、晶圆级封装、2.5D/3D 封装。

AI 传导：AI GPU/ASIC、HBM、先进 DRAM、先进逻辑扩产 → 光刻步骤增加、先进节点需求提升 → KrF、ArF、EUV、先进封装光刻胶需求提升。

## 2. A 股

彤程新材：通过北京科华承载 IC 光刻胶，A 股光刻胶平台型核心标的。

南大光电：ArF 光刻胶国产突破核心标的，技术突破意义强，重点看量产放量。

晶瑞电材：g/i 线、KrF 光刻胶重要标的，ArF 研发推进中，兼具湿电子化学品平台属性。

上海新阳：KrF、ArF 光刻胶研发验证和产业化推进，同时有电镀液、清洗液等半导体材料。

艾森股份：先进封装光刻胶、PSPI、功能湿法材料，与 HBM、Chiplet、先进封装关系更强。

华懋科技：通过徐州博康间接映射光刻胶单体、树脂、PAG 和成品胶。

容大感光：g/i 线、PCB/显示感光材料，高端 IC 光刻胶纯度较低。

## 3. 美股

Qnity Electronics：光刻胶和光刻材料核心标的，承接原 DuPont 电子材料体系，覆盖 EUV、ArF、KrF、i-line 等。

Entegris：光刻材料配套、过滤、污染控制，不是成品光刻胶主线。

JSR / Inpria：产业核心但已私有化退市，不是可直接交易标的。

## 4. 韩股

Dongjin Semichem：韩股光刻胶第一核心，ArF、KrF、i-line 和配套材料布局完整。

Samyang NC Chem：光刻胶核心原料、PAG、polymer、KrF/ArF/EUV 材料。

ENF Technology：光刻配套化学品，thinner、stripper、cleaner 等。

## 5. 日股

Tokyo Ohka Kogyo：全球高端光刻胶核心龙头之一，EUV、ArF、KrF 产品线完整。

Shin-Etsu Chemical：半导体材料超级龙头，光刻胶是核心电子材料之一。

Fujifilm：光刻胶 + 湿电子化学品 + CMP 材料平台。

Sumitomo Chemical：ArF 浸没式和 EUV 光刻胶核心标的。

Mitsubishi Chemical：光刻胶原料和电子材料平台观察。

JSR：全球光刻胶龙头级公司，但已退市，只作为产业参考。

一句话总结：

**光刻胶全球核心在日本产业链，重点看 TOK、信越、Fujifilm、住友化学和未上市 JSR；美股看 Qnity；韩股看 Dongjin 和 Samyang；A 股看彤程、南大、晶瑞、上海新阳，先进封装光刻胶看艾森股份。**
