# 5.7 封装基板

封装基板，也叫 IC 载板，是芯片封装里的关键承载平台，位于芯片和 PCB 主板之间。它负责保护芯片、承载芯片，并把芯片内部极细密的电信号和电源连接转换成外部主板可以连接的线路。

一句话理解：

**封装基板负责把芯片和外部系统连接起来，是高端芯片从裸片变成可用器件的关键桥梁。**

## 1. 产品分类

BT 载板：存储、射频、CSP、SiP、消费电子和成熟封装。  
ABF 载板 / FC-BGA：高性能 CPU、GPU、FPGA、ASIC、AI 加速器最重要的封装基板路线。  
FC-CSP 载板：手机 AP、移动芯片和部分高集成芯片封装。  
存储类封装基板：DRAM、NAND、HBM 相关封装。  
玻璃基板：下一代先进封装潜在路线，优势是大尺寸、低翘曲、平整度和高密度互连。

AI 传导：AI GPU/ASIC、HBM、Chiplet、CoWoS、2.5D/3D 封装扩张 → 单颗封装面积变大、层数增加、线宽线距变细、供电和高速信号要求提高 → ABF/FC-BGA、高端 BT、玻璃基板和封装基板材料需求提升。

## 2. A 股

兴森科技：A 股封装基板国产替代核心，CSP、FC-BGA、ABF、玻璃基板布局。

深南电路：PCB + 封装基板平台龙头，中国封装基板领域先行者。

生益科技：封装基板用基板材料 / 电子电路基材龙头，不是封装基板制造商。

珠海越亚：国内 IC 封装载板重要公司，未上市 / IPO 观察。

## 3. 美股

美股暂无纯正 ABF / FC-BGA 封装基板主板龙头。TTM Technologies 可作为先进 PCB / substrate-like PCB 低纯度观察。

## 4. 韩股

Samsung Electro-Mechanics：韩股 FC-BGA / AI 服务器封装基板第一核心标的。

LG Innotek：FC-BGA 成长型标的，从移动基板向 AI / 数据中心升级。

Daeduck Electronics：韩国 IC 封装基板重要标的，PKG、FC-CSP、CSP。

Simmtech：存储封装基板和 memory module PCB 核心标的。

Korea Circuit：FCBGA、FCCSP、SiP、CSP 封装基板补充。

SKC：玻璃基板方向，通过 Absolics 映射，传统载板纯度不高但下一代基板弹性强。

## 5. 日股

Ibiden：全球高端 ABF / FC-BGA 封装基板核心龙头。

Ajinomoto：ABF 膜核心供应商，不做基板但卡在关键材料环节。

TOPPAN：FC-BGA 封装基板重要标的。

Kyocera：有机封装基板和陶瓷封装重要标的。

Sumitomo Bakelite：封装基板材料，core/prepreg/低翘曲材料。

Panasonic：封装基板材料和电子材料补充。

Meiko：PCB + 半导体封装基板补充标的。

Shinko Electric：产业重要但已退市。

## 6. 范围外但必须知道

Unimicron、Kinsus、Nan Ya PCB：台股 ABF / IC 载板重要公司。  
AT&S：欧洲先进 IC substrate 核心公司之一。

一句话总结：

**封装基板制造看 Ibiden、三星电机、兴森科技、深南电路、TOPPAN、LG Innotek；ABF 膜看 Ajinomoto；材料看生益科技、Sumitomo Bakelite；玻璃基板看 SKC / Absolics。**
