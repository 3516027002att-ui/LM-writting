# 5.4 靶材

靶材是 PVD 溅射工艺里的核心材料。PVD 设备通过等离子体轰击靶材，把靶材表面的原子打出来，再沉积到晶圆表面，形成金属层、阻挡层、种子层、透明导电层等薄膜。

一句话理解：

**靶材负责把金属或化合物材料沉积到晶圆上，是 PVD 成膜的材料源头。**

## 1. 产品分类

半导体 IC 靶材：铜、铝、钛、钽、钨、钴、镍铂、铜锰合金等。  
显示 / 光伏靶材：ITO、钼、IGZO、AZO 等。  
贵金属靶材：金、铂、钯等。  
高纯金属材料：高纯金属提纯、晶粒控制、织构控制、热等静压、焊接、加工、清洗和回收。

AI 传导：AI GPU/ASIC、HBM、先进 DRAM、3D NAND、先进逻辑扩产 → PVD 金属沉积步骤增加或规格提高 → 高纯金属靶材需求提升。

## 2. A 股

江丰电子：A 股半导体高纯金属溅射靶材第一核心标的，先进逻辑和先进存储靶材国产替代主线最直接。

有研新材：通过有研亿金布局 IC 靶材和高纯金属材料。

阿石创：PVD 镀膜材料重要标的，显示、光伏、光学靶材更强，半导体 IC 靶材纯度低于江丰电子。

隆华科技：钼靶材、ITO 靶材，主要偏显示和光伏。

## 3. 美股

Materion：高纯金属靶材和贵金属靶材核心标的，重点看钽靶、贵金属靶材和先进半导体材料需求。

Solstice Advanced Materials：半导体材料平台，靶材业务来自原 Honeywell 先进材料体系。

Linde：气体龙头中的靶材相关补充，产业意义有但投资纯度低。

## 4. 韩股

韩股暂无纯半导体 IC 高纯金属靶材上市龙头。Advanced Nano Products 可作为 ITO/透明导电靶材观察标的，更偏显示和光伏。

## 5. 日股

JX Advanced Metals：日股半导体靶材第一核心标的，全球高纯金属溅射靶材龙头级公司。

Tosoh：半导体和显示靶材重要供应商。

ULVAC：PVD 设备 + 半导体靶材相关标的。

Sumitomo Chemical：铝靶材重要标的。

Sumitomo Metal Mining：高纯金属、贵金属、稀有金属靶材相关。

Mitsui Kinzoku：ITO/IGZO 等显示和透明导电靶材。

一句话总结：

**半导体 IC 靶材核心看 JX Advanced Metals、江丰电子、Materion、Tosoh、有研新材；显示/光伏靶材看阿石创、隆华科技、Advanced Nano Products、Mitsui Kinzoku。**
