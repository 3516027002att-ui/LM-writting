# 5.1 硅片

硅片是半导体制造最基础材料，是所有晶体管、电路结构、存储单元和互连层的地基。

## 1. 产品分类

抛光片：基础晶圆。  
外延片：在硅片表面生长高质量单晶硅外延层，常用于功率、模拟、CIS、部分逻辑和车规。  
SOI 硅片：绝缘体上硅，用于射频、低功耗、传感器、硅光。  
300mm 硅片：先进逻辑、先进存储、大规模晶圆厂主流规格。  
200mm 及以下：功率、模拟、MEMS、分立器件、成熟制程。

AI 传导：AI GPU/ASIC、HBM、先进 DRAM、先进逻辑、数据中心电源芯片需求提升 → 晶圆厂扩产和先进工艺需求提升 → 300mm 高端硅片、外延片、SOI 需求提升。

## 2. A 股

沪硅产业：A 股 300mm 半导体硅片国产替代核心标的，子公司新昇是关键资产。

立昂微：半导体硅片 + 功率器件 + 化合物半导体一体化，偏功率、模拟、车规、射频和成熟制程。

有研硅：半导体硅抛光片、刻蚀设备用硅材料、硅零部件等。

TCL 中环：硅材料平台，核心偏光伏硅片，半导体材料有布局但受光伏周期影响大。

神工股份：半导体级单晶硅材料、刻蚀用单晶硅材料和硅零部件。

## 3. 美股

暂无特别纯正的美股主板硅片龙头。Coherent 更偏 SiC 衬底/外延，不属于传统硅片主线。

## 4. 韩股

SK Siltron 是韩国硅片产业核心，但未独立上市，可通过 SK Inc 间接观察，纯度较低。

## 5. 日股

Shin-Etsu Chemical：全球硅片绝对龙头级公司，产品包括抛光片、扩散片、外延片、SOI 片、退火片等。

SUMCO：高纯半导体硅片纯标的，半导体硅片周期和 300mm 高端硅片需求主线清晰。

## 6. 范围外但必须知道

GlobalWafers：台股，全球硅片核心龙头。  
Siltronic：德股，欧洲硅片核心公司。

一句话总结：

**硅片全球核心看 Shin-Etsu、SUMCO、GlobalWafers、Siltronic、SK Siltron；A 股看沪硅产业、立昂微、有研硅。**
