# 5.6 CMP 材料

CMP 材料是化学机械抛光过程中的核心耗材。CMP 设备负责执行抛光动作，但真正决定抛光效果的是抛光液、抛光垫和 CMP 后清洗材料。

一句话理解：

**CMP 设备负责怎么磨，CMP 材料决定磨得快不快、平不平、伤不伤晶圆。**

## 1. 产品分类

抛光液：由磨粒、化学添加剂、氧化剂、络合剂、腐蚀抑制剂组成，决定去除速率、选择比、缺陷率和表面质量。

抛光垫：晶圆和抛光头之间的工作界面，决定压力分布、浆料流动、摩擦稳定性和缺陷控制。

CMP 后清洗液：去除颗粒、金属残留、有机残留和腐蚀副产物，防止后续工艺污染。

AI 传导：AI GPU/ASIC、HBM、先进 DRAM、3D NAND、先进逻辑、先进封装扩产 → 多层互连、TSV、RDL、金属层和介质层更复杂 → CMP 步骤增加，材料体系升级。

## 2. A 股

安集科技：A 股 CMP 抛光液第一核心，国产替代主线最直接。

鼎龙股份：A 股 CMP 抛光垫第一核心，并向抛光液和清洗液延伸。

上海新阳：CMP 后清洗和功能湿电子化学品观察，先进封装湿法材料相关性强。

## 3. 美股

Qnity Electronics：全球 CMP 抛光垫强标的，同时有 CMP slurry，承接原 DuPont 电子材料体系。

Entegris：CMP 材料 + 污染控制 + post-CMP 平台型标的。

3M：CMP 抛光垫和 Pad Conditioner 补充标的，整体业务纯度较低。

## 4. 韩股

KCTech：CMP 设备 + CMP slurry，韩国存储链条相关。

Soulbrain：CMP slurry、高纯化学品、清洗刻蚀材料。

Dongjin Semichem：电子材料平台，CMP slurry 是 HBM/存储延伸。

SKC：CMP pad 历史相关，但业务出售后纯度需核实。

## 5. 日股

Fujimi：日股 CMP slurry 第一核心之一，精密研磨材料主线强。

Resonac：半导体材料平台，CMP slurry 是重要前道材料。

Fujifilm：CMP slurry 和 post-CMP cleaner 协同强，先进封装相关性好。

AGC：大型材料集团，CMP slurry 补充观察。

Kuraray：CMP pad 补充观察，poreless pad 有辨识度。

一句话总结：

**CMP 抛光液看 Fujimi、安集、Resonac、Fujifilm、Soulbrain、Dongjin；CMP 抛光垫看 Qnity、鼎龙、Kuraray；CMP 后清洗和污染控制看 Entegris、Fujifilm、上海新阳。**
